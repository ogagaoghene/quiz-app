# Quiz App 

## Tech Stack

**Client:** HTML, CSS, and JavaScript

## Run Locally

Clone the project

```bash
  git clone https://github.com/ogagaoghene/quiz-app.git
```

Go to the project directory

```bash
  cd quiz-app
```

Launch the quiz

```bash
  double click on index.html 
  or 
  Click on a live-server on vscode 
```
